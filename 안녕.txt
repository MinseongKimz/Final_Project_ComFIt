Java Bean
	자바빈은 자바 클래스이므로 자바 클래스를 설계하는 것과 기본 규칙이 같다
	자바 빈에서는 멤버 변수(전역 변수)를 프로퍼티라고 부른다.
	클래스 선언은 public , 프로퍼티 선언은 private 으로 한다.
	(매개 변수가없는) 기본 생성자를 포함한다.
	직렬화 할 수 있어야 한다.
JDBC DRIVER
	드라이버는 데이터베이스에 대한 연결을 제공하고 클라이언트와 데이터베이스 간에 
	쿼리 및 결과를 전송하기 위한 프로토콜을 구현한다.

	MVC
	Model 은 로직을 가지고 있는 부분으로 DB와 연동
	View는 사용자에게 제공하는 화면으로 UI에 해당
	Controller는 View 와 Model 사이에 흐름을 제어하는 개념

	Jsp:useBean
	id : 객체 인스턴스를 식별하는 이름으로 대소문자 까지 엄격하게 구분한다.
	한 번 생성된 Bean이 소멸할 때 까지는 같은 Id를 중복해서 사용할 수 없다.
	
	Scope: 객체 참조 범위는 기본값(default)은 page이다.

	.page
	객체 참조는 service() 호출이 종료될 때 소멸.
	
	.request
	HttpServletRequest 객체가 소멸되지 않는 한 존재
	컨테이너에 의해 생성된 서블릿은
	HttpServletRequest의 의 setattribute()메소드를 이용하며
	객체를 HttpServletRequest 에 바인딩시킨다.
